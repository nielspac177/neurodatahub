════════════════════════════════════════

      claude

2) Pega esto en Claude Code:

3) Cuando te enseñe drafts/needs_human/, decide (ver más abajo).

4) Para publicar, dile:  "haz push"

════════════════════════════════════════
LO ÚNICO QUE DECIDES TÚ (drafts/needs_human/)
════════════════════════════════════════

Cada archivo trae el motivo. Los típicos:

  "'open' con una sola derivación"
      Abre la URL. ¿Se descarga sin cuenta?
      Sí -> publicar.   No -> corregir el acceso.

  "el id ya existe"        ya está en el catálogo. Descartar.
  "N citas no verificables" el modelo se inventó frases. Descartar.
  "falta campo obligatorio" completar a mano o descartar.

REGLA DE ORO: si dudas del acceso, NO lo publiques como abierto.
Equivocarse ahí le cuesta semanas a un estudiante.


════════════════════════════════════════
PUBLICAR
════════════════════════════════════════

   cd ~/neurodatahub
   git checkout main
   git merge <la-rama-que-te-dio>
   git push

El push despliega el sitio (~1 min). Antes de eso no hay nada
publicado. Si al abrir el sitio los filtros no filtran tras un
update, recarga fuerte una vez: Cmd+Shift+R.


════════════════════════════════════════
COSTE (importante)
════════════════════════════════════════

Un lote de enriquecimiento cuesta bastante (millones de tokens),
todo en la corrida mensual. Por eso:

  - Empieza SIEMPRE con --limit 3 y mira /usage.
  - Si queda presupuesto, sube el lote (--limit 10, 20).
  - No hace falta procesar los miles de candidatos en cola: los
    mejores están arriba; el triaje ya descartó el resto.
